# AdaptixC2 v1.0 中文版

12月30日：[v1.0版本更新了什么？](https://adaptix-framework.gitbook.io/adaptix-framework/changelog-and-updates/version-1.0)

Adaptix 是一个可扩展的渗透后利用和对抗性仿真框架，专为授权渗透测试而设计。Adaptix 服务器使用 Golang 编写，以提供操作灵活性。GUI 客户端使用 C++ QT 编写，可以在 Linux、Windows 和 MacOS 操作系统上使用。[完整文档请访问这里](https://adaptix-framework.gitbook.io/adaptix-framework)。

## 法律警告

本工具仅用于授权的安全测试和红队行动。未经授权的使用严格禁止，可能违反当地和国际法律。使用风险自负。



## 快速开始

请查看 [wiki文档](https://adaptix-framework.gitbook.io/adaptix-framework/adaptix-c2/getting-starting/installation)。



## 功能特性
* 服务器/客户端架构支持多人协作
* 跨平台图形界面客户端
* 完全加密通信
* 监听器和代理作为插件（扩展器）
* AxScript 脚本引擎
* 任务和作业存储
* 凭证管理器
* 目标管理器
* 远程终端 / Shell
* 文件和进程浏览器
* Socks4 / Socks5 / Socks5 身份验证支持
* 本地和反向端口转发支持
* BOF 支持
* 代理和会话关系图
* 代理健康检查器
* 代理终止日期和工作时间控制
* Windows/Linux/MacOS 代理支持
